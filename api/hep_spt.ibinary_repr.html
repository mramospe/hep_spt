ibinary_repr¶

hep_spt.ibinary_repr(arg)[source]¶

Get the binary representation of the given value(s). This function is equivalent to numpy.binary_repr(), but the returned value is an integer.

Parameters:arg (int or numpy.ndarray(int)) – array of values.
Returns:Values in binary representation (as integers).
Return type:numpy.ndarray(int)
Raises:TypeError – If the input does not contain integer values.
