wald_int¶

hep_spt.wald_int(k, N, cl=0.6826894921370859)[source]¶

Parameters

  • k (int or numpy.ndarray(int)) – passed events.

  • N (int or numpy.ndarray(int)) – total number of events.

  • cl (float or numpy.ndarray(float)) – confidence level.

Returns
