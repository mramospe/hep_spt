residual¶

hep_spt.residual(vals, err, ref, ref_err=None)[source]¶

Calculate the residual values from “vals” with respect to “ref”. If the uncertainties are also provided for the reference, the errors are recalculated, using the sum of squares rule. Being \(v\) the value and \(r\) the reference, this translates into:

\((\sigma^{v}_{low})^2 = (\sigma^{v}_{low})^2 + (\sigma^{r}_{up})^2\)

\((\sigma^{v}_{up})^2 = (\sigma^{v}_{up})^2 + (\sigma^{r}_{low})^2\)

Parameters:
  • vals (array-like) – values to compare with.
  • err (array-like) – array of errors. Both symmetric and asymmetric errors can be provided. In the latter case, they must be provided as a (2, n) array.
  • ref (array-like) – reference to follow.
  • ref_err (array-like) – possible errors for the reference.
Returns:

residual of the values with respect to the reference and associated errors. In case asymmetric errors have been provided, the returning array has shape (2, n).
Return type:

array-like, array-like
Raises:

TypeError – if any of the error arrays does not have shape (2, n) or (n,).
