remote_protocol¶

hep_rfm.remote_protocol(a, b)[source]¶

Determine the protocol to use given two paths to files. Return None if the two protocols are not compatible.

Parameters

  • a (str) – path to the firs file.

  • b (str) – path to the second file.

Returns

protocol ID.

Return type

int or None
