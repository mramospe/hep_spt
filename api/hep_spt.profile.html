profile¶

hep_spt.profile(x, y, bins=20, range=None)[source]¶

Calculate the profile from a 2D data sample. It corresponds to the mean of the values in “y” for each bin in “x”.

Parameters:
  • x (collection(value-type)) – values to consider for the binning.
  • y (collection(value-type)) – values to calculate the mean with.
  • bins (int or sequence of scalars or str) – see numpy.histogram().
  • range (tuple(float, float)) – range to process in the input array.
Returns:

profile in “y”.
Return type:

numpy.ndarray
