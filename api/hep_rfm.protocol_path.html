protocol_path¶

hep_rfm.protocol_path(path, protocol=None)[source]¶

Return a instantiated protocol using the given path and protocol ID. If None is provided for “protocol”, then a LocalPath will be used.

Parameters

  • path (ProtocolPath) – path to process.

  • protocol (str) – protocol ID to use.

Returns

protocol associated to the given path.

Return type

ProtocolPath
