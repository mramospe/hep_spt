FlatDistTransform¶

class hep_spt.FlatDistTransform(points, values=None, kind='cubic')[source]¶

Bases: object

Instance to transform values following an unknown distribution \(f(x)\) into a flat distribution. This class takes into account the inverse transform sampling theorem, which states that, given a distribution \(f(x)\) where \(x\in[a, b]\) then, given a random variable following a flat distribution r,

\[F(x) - F(x_0) = \int_{x_0}^x f(x) dx = \int_0^r r dr = r\]

where \(F(x)\) is the primitive of \(f(x)\). This allows us to generate values following the distribution \(f(x)\) given values from a flat distribution

\[x = F^{-1}(r + F(x_0))\]

In this class, the inverse process is performed. From a given set of values of a certain distribution, we build a method to generate numbers following a flat distribution.

The class performs an interpolation to get the transformated values from an array holding the cumulative of the distribution. The function scipy.interpolate.interp1d() is used for this purpose.
