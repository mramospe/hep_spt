process_range¶

hep_spt.process_range(arr, rg=None)[source]¶

Process the given range, determining the minimum and maximum values for a 1D histogram.

Parameters:
  • arr (numpy.ndarray) – array of data.
  • rg (tuple or None) – range of the histogram. It must contain tuple(min, max), where “min” and “max” can be either floats (1D case) or collections (ND case).
Returns:

minimum and maximum values.
Return type:

float, float
