CLsTS_continuous¶

class hep_spt.CLsTS_continuous(alt, null)[source]¶

Bases: hep_spt.CLsTS

Base class to represent the test-statistics function to work with the CLs method using hypothesis working on a continuous domain. The class is built from the alternative and null hypotheses.

Parameters

  • alt (CLsHypo) – alternative hypothesis.

  • null (CLsHypo) – null hypothesis.

Variables

  • ah – alternative hypothesis.

  • nh – null hypothesis.
